# 수련회 조편성

## 입력
- 참석자: `participant_id`, `public_name`, `campus`, `grade_band`, `gender`, `engagement_score`, `newcomer`, `attendance_slots`.
- 관계: 함께 배정 선호, 분리 필요, 리더 후보, 수동 고정 조.
- 설정: 조 수, 목표 크기, 항목별 가중치.

## 절차
1. 식별자 중복과 필수값 누락을 검사한다.
2. 수동 고정 배정을 먼저 적용한다. 운영자의 판단이 자동 점수보다 우선하기 때문이다.
3. 각 후보 배치의 조 크기, 학년·소속·성향 분산, 새친구·리더 분포와 관계 제약을 점수화한다.
4. 점수가 가장 낮은 불균형을 줄이는 방향으로 반복 배정한다.
5. 인접 조 교환으로 총 불균형 점수를 개선하되 최대 반복 횟수를 둔다.
6. 결과와 함께 경고, 수동 확인 항목, 배정 이유를 출력한다.

## 불변조건
- 한 참석자는 활성 조 하나에만 속한다.
- 불참자는 배정하지 않는다.
- `locked_group`은 자동 실행이 변경하지 않는다.
- 충족 불가능한 관계 제약은 숨기지 않고 경고한다.

## 출력
- 운영용: 전체 점수·경고·수동 변경 이력.
- 공개용: `public_name`, `group_name`, 선택적 `role`만 포함.

## 테스트 프롬프트
1. "참석자 48명을 6개 조로 나누되 임동/수완과 중·고등 비율을 맞춰줘."
2. "새친구는 각 조에 분산하고 리더 후보는 조마다 1명 이상 배치해줘."
3. "3조 두 명은 고정한 채 나머지만 다시 편성해줘."
