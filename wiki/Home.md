# 2026 여름수련회 조편성·차량 운행 안내

수련회 조편성과 광주 ↔ 수련회장 차량 운행은 아래 공개 안내 페이지에서 확인합니다.

## 바로 확인하기

**[조편성·차량 운행 안내 페이지 열기](https://osh1993.github.io/26summerschool/)**

안내 페이지에서는 다음 정보를 검색할 수 있습니다.

- 날짜와 방향(IN/OUT)별 차량 운행
- 차량별 남은 승객 좌석과 운행 상태
- 공개 참가자 코드별 탑승 차량
- 공개 참가자 코드별 조편성

## 현재 게시 상태

이 Wiki는 JavaScript를 실행하지 않는 정적 안내 문서입니다. 최신 조편성·차량 배치는 위 GitHub Pages 안내 페이지를 기준으로 확인해 주세요.

현재 저장소의 화면 데이터는 기능 검증용 **합성 예시**입니다. 실제 운영 데이터 연결 후 안내 페이지 상단의 `최신 공시 정보` 표시와 공시 시각을 확인해야 합니다.

## 개인정보 보호

- 공개 페이지에는 실명, 전화번호, 생년월일, 보호자 연락처를 게시하지 않습니다.
- 전달받은 참가자 공개 코드로만 정보를 찾습니다.
- 참가자 코드와 실제 이름의 대응 관계를 Wiki에 쓰지 마세요.
- 집결지 상세주소와 개인 연락 정보는 별도의 비공개 안내 수단으로 전달합니다.
