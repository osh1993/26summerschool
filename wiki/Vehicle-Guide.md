# 차량 운행 확인 방법

상세 운행표는 [조편성·차량 운행 안내 페이지](https://osh1993.github.io/26summerschool/)에서 확인합니다.

## 방향 표시

- **IN · 수련회장으로**: 광주에서 수련회장으로 이동하는 운행
- **OUT · 광주로**: 수련회장에서 광주로 이동하는 운행

## 내 차량 찾기

1. 안내 페이지에서 `차량 운행` 탭을 엽니다.
2. 날짜와 방향을 선택합니다.
3. 운영자에게 전달받은 참가자 코드(예: `A-01`)를 검색합니다.
4. 표시된 출발 시각, 집결지 라벨, 차량과 운행 상태를 확인합니다.
5. `공개 탑승 코드 확인`을 펼쳐 자신의 코드를 다시 확인합니다.

## 상태 의미

| 표시 | 의미 |
|---|---|
| 접수 중 | 아직 최종 확정 전이며 배치가 바뀔 수 있음 |
| 운행 확정 | 공시된 시각과 차량으로 운행 예정 |
| 출발함 | 차량이 집결지에서 출발함 |
| 도착함 | 차량이 목적지에 도착함 |
| 운행 취소 | 해당 운행을 이용하지 말고 운영자 안내 확인 |

`전체 정원`은 운전자를 포함한 차량 정원입니다. `남은 승객 좌석`은 운전자 좌석을 제외하고 추가로 탑승할 수 있는 인원입니다.

## 주의사항

- 공개 장소 라벨만 표시되므로 상세 집결 위치는 비공개 공지를 확인하세요.
- 화면 상단이 `마지막 정상 공시본`이면 실시간 연결이 되지 않은 상태일 수 있습니다.
- 화면 상단이 `샘플 데이터`이면 실제 배정이 아닙니다.
- 변경이 예상되면 페이지의 마지막 공시 시각을 확인하세요.
